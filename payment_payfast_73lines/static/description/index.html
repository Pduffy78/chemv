PayFast is a payments processing service for South Africans and South African websites. PayFast enable easy, secure and instant transfer of money from online buyers to sellers.This modules allows to make payment using PayFast.

Get Credentials

PayFast provides test credentials which can be used to test payfast payment gateway. Login to https://sandbox.payfast.co.za and you will get Merchant ID and Merchant Key.
PayFast currently supports only ZAR currency.

Acquirer

On the payment page, select the PayFast Payment Acquirer and click on Pay Now button.

Make Payment

After clicking on Pay Now, you will be redirected to PayFast payment page. Select the payment method you want to make payment(normally wallet) with and make the payment.

Order Confirmed

Once the Payment is made the order is confirmed.

Order in PayFast Dashboard

The order for which the payment is made appears in the dashboard of PayFast. But for Test environment it is not shown.
